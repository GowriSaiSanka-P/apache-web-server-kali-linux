# Apache2 Web Server Setup on Kali Linux

## Project Overview

This project demonstrates how to install and configure an Apache2 web server on Kali Linux, host a custom HTML webpage, monitor Apache access logs, and manage access using UFW firewall rules.

## Technologies Used

- Kali Linux
- Apache2
- HTML
- UFW Firewall
- Linux Networking

## Features

- Installed Apache2 web server
- Hosted custom HTML webpage
- Accessed webpage using local IP
- Analyzed Apache access logs
- Configured firewall allow/deny rules
